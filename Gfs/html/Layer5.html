    Voc.:

  1. establishing = refers to the process of creating or setting up a connection or communication between two or more devices
  2. maintenance = process of keeping things in good conditions

The Session layer is the fifth layer of the OSI model.
It is responsible for establishing, managing, and terminating communication sessions
between applications on different devices. The Session layer provides services
such as session establishment, checkpointing, and session termination.
It also provides synchronization between different applications,
allowing them to communicate with each other without interference.
The Session layer is critical for ensuring that data is transmitted correctly between applications
and that sessions are terminated properly to prevent resource waste.
