  1. logical addressing = a method of identifying devices
  2. maintenance = process of keeping things in good conditions
  3. routing = process of selecting a path for network traffic
  4. congestion = traffic jam for networking

The Network layer is the third layer of the OSI model.
It is responsible for the logical addressing and routing of data over the network,
including the creation and maintenance of routing tables,
and the selection of the best path for data to travel. The Network layer uses logical addresses,
such as IP addresses, to identify devices on the network, and it uses routing protocols
to determine the best path for data to travel from the source to the destination.
The Network layer also provides congestion control and packet sequencing
to ensure that data is transmitted reliably over the network.
